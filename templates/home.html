TIBCO Mapping Generator

Export the mapping defined in TIBCO BW 5.x processes

This application provides APIs that can be used to export the mapping defined in the TIBCO BW 5.x processes.

  • The scope is to extract the mapping content of the Map Data activity only.
  • The response can be copied to excel in tabular form.
  • Basic exposure to xml/xslt is required to understand and interpret the mapping exported to the table.

API Details

API URI https://tibcomappinggenerator.herokuapp.com/generatetable
API HTTP Method POST,GET (both work the same for ease of use)
Content Type Doesn't matter. This is not required.
API Input TIBCO BW 5.x process definition xml content in full(.process file content)
API Response Tabular format of the mappings defined all the Map Data activities.
How to use Use from any REST API invocation tools. eg : Postman, SOAPUI, JMeter. You can even invoke from your JS,Java,Python code.

Use cases and benefits

  • You simply want to see the tabular view of your mapping.
  • To explain the mapping rules to someone.
  • Generate a (sort of) mapping sheet from the actual code.
  • This export will contain the actual mapping rules and will be way better than any excel sheet representation.
  • Can be used for cross validation with your mapping sheets.
  • Can be generated anytime.
  • Helpful when mapping has several loops and conditional mapping.
  • Extermely helpful in dealing with huge schemas like IDocs.
  • Can be searched easily.
